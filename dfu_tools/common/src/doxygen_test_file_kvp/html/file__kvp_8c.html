Detailed Description

Parses a KVP file and passes the parsed KVP back to the caller for each line it processes.

Callers can walk through each line of a KVP file, parsing each and then use the parsed data. They can also do a higher-level "search" operation that can walk the entire file to find a value associated with a specified key.
